Kneuron 1.0.0 - Keyboard Shortcuts

Global Shortcuts

Enter Save or submit form
Works on any form or editing view
Esc Close or cancel
Closes modals or cancels current action

Navigation (with Alt)

Alt + 1-6 Navigate main menu
Quick access to main navigation items on 1st column of the page
Alt + ` (backtick) Go back
Click on left arrow at top to return to previous sub-section

View Settings Tools (with Alt)

These will only work when a view is selected

Alt + S Settings
Quick access to view settings' menu
Alt + Q, A, Z or X 1st to 4th Tool
Quick access to the first to fourth tool in 2nd column of the page
